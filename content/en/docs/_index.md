{{% pageinfo %}}
Klustre CSI Plugin is an open-source [Container Storage Interface (CSI)](https://kubernetes-csi.github.io/docs/) node driver that lets Kubernetes workloads mount existing Lustre file systems. It targets high-throughput, shared `ReadWriteMany` use cases such as HPC, AI/ML training, and media workloads.
